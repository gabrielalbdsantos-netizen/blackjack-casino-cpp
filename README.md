# Blackjack Simulator em C++17

Este projeto consiste em um simulador completo e interativo do clássico jogo de cassino **Blackjack (Vinte e Um)** desenvolvido em **C++17** para execução via terminal. A aplicação foca na arquitetura de regras de negócios, controle de fluxo complexo, inteligência artificial baseada em tomada de decisão e gerenciamento de estado (sistema de apostas/fichas).

## 🚀 Como Executar o Projeto

Certifique-se de ter um compilador com suporte a C++17 instalado (como o GCC/TDM-GCC). No terminal, execute os seguintes comandos:

```bash
# Compilação utilizando a flag do padrão C++17
g++ -std=c++17 -o blackjack blackjack.cpp

# Execução do binário gerado
./blackjack
